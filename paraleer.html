Que es Seo

Search Engine Optimization: Es el proceso de conseguir visitas a un sitio web en los resultados de búsqueda de los navegadores

Pasos para hacer Seo
  • Definir nombre del dominio, extensión del dominio, búsquedas de palabras clave
  • Hacer un análisis de las palabas clave que las gente busca
  • Ganar mas tráfico con tu página de inicio posicionando palabras clave potentes y difíciles
  • Crear una capa Seo con las etiquetas h3,h2,h1 para los títulos, no usar la etiqueta de párrafo para los titulos
  • Identificar las palabras clave y grupos de palabras clave
  • Revisar las palabras clave y la semantica que utiliza la competencia
  • Se debe analizar y hacer cambios que no mofiquen el lenguaje ni el mensaje del texto, que permita a google entender que esa web o esa url se tiene que posicionar por mas palabras clave
